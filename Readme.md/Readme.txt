# TV247 Repository

This repository hosts:

- **repository.tv247** — the Kodi repository addon  
- **plugin.program.tv247** — the TV247 program addon  
- Optional builds available for download

---

## Build Download

You can download the full Kodi build (dab19) here:

[Download dab19.zip from Dropbox](https://www.dropbox.com/scl/fi/90rsb9oal9dc3fp3g1l8s/dab19.zip?rlkey=5st59x4bq5xpvljnf0rlflu1z&st=h52ojqnq&dl=1)

> Note: This build is separate from the repository and plugin addons. It is a full Kodi build.

---

## Installing the Repository in Kodi

1. Download the **repository zip** (`repository.tv247-1.0.0.zip`) from the repository folder or GitHub releases.  
2. Open Kodi → `Settings → Add-ons → Install from zip file`.  
3. Select the zip you downloaded.  
4. The repository will be installed.  
5. Go to `Install from repository → TV247 Repository` to install `plugin.program.tv247` or other addons.
